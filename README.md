# AI Prospect Research Agent

An AI-powered prospect research workflow built with Relay.app.

---

## Features

### Research Layer
- Google search
- Website crawling
- Research consolidation

### Intelligence Layer
- Company extraction
- Competitor discovery
- Market analysis
- Pain point analysis
- Tech stack detection
- Automation opportunities

### Scoring Layer
- ICP matching
- Lead scoring
- Priority classification

### Outreach Layer
- Personalized cold email
- Follow-up sequences
- Proposal generation

### Reporting Layer
- Executive report generation
- Google Sheets logging
- High-priority lead alerts

---

## Workflow Architecture

```
Company Name + Website
        ↓
Google Search
        ↓
Website Crawling
        ↓
Research Consolidation
        ↓
Company Intelligence
        ↓
Competitor Analysis
        ↓
Pain Point Detection
        ↓
Lead Scoring
        ↓
Personalized Outreach
        ↓
Executive Report
        ↓
Google Sheets
```
